## Функционал (стабильный)

### Задачи (CRUD)
- **Поля**: название, описание, статус (7 шт.), приоритет (5 шт.), тип (4 шт.), срок сдачи (date input с календарём), комментарий, sprintId
- **Статусы**: Новая / На уточнении / Аналитика / В работе / QA / Ревью / Готово — цветные подложки
- **Приоритеты**: Критичный (чёрный), Высокий (красный), Средний (жёлтый), Низкий (синий), Очень низкий (зелёный) — полупрозрачные 50%
- **Типы**: Баг / Фича / Обновление / Рутина — цветные бейджи
- **ID**: порядковый номер (1, 2, 3...), auto-increment
- **inline-редактирование**: статус (select) и срок (date) прямо из таблицы
- **Комментарий**: текстовое поле в таблице, при наведении — тултип с полным текстом

### Спринты (CRUD)
- **Поля**: название, дата начала/окончания
- **Статусы**: ожидает → начать (становится активным) → завершить (completed)
- **Завершение спринта**: все задачи должны быть в статусе «Готово», иначе ошибка
- **Архив**: завершённые спринты сворачиваются под «📦 Архив»

### Навигация (левая панель)
1. **🔥 Срочные** — дедлайн сегодня/завтра, статус не «Готово» (счётчик)
2. **👤 Мои таски** — ручной список задач (☆ добавить / ★ убрать)
3. **📋 Бэклог** — задачи без спринта
4. **Активные спринты** — pending и active
5. **📦 Архив** — completed спринты (сворачивается)
6. Внутри «Мои таски» и спринтов: **табы «Активные» / «Закрытые»**

### Чекбоксы и массовые операции
- **Чекбокс** у каждой задачи, **Select All** в шапке
- Панель при выборе: счётчик, список спринтов, кнопка «Перенести», «Снять выделение»

### Фильтры и сортировка
- **Статус**: чип-мультиселект (дропдаун с чекбоксами, чипы с ✕, поиск внутри)
- **Приоритет / Тип**: обычный select
- **Поиск**: по ID или названию
- **Сортировка**: по всем колонкам (клик по заголовку)
- **Пресеты фильтров**: сохранение/применение

### Drag-and-drop
- Задачи перетаскиваются в спринты, бэклог, мои таски
- Нельзя в завершённый спринт

### Отображение по умолчанию
- **Срочные задачи** при загрузке (розовая подсветка)
- Кнопка «Все задачи» / «Срочные»
- Счётчики показывают **только активные задачи** (без «Готово»)

### Заметки (левая панель)
- CRUD заметок, **auto-resize** (hug), фокус на новой
- Сохраняются в localStorage и Firebase

### Бэкапы
- Звуковой сигнал раз в час → confirm → создаётся бэкап
- Хранится **3 последних** копии
- Восстановление, экспорт/импорт JSON в настройках (⚙️)

### Синхронизация
- **Firebase Firestore** (коллекция `siberia_tracker`)
- **Автосинхронизация** каждые 5 секунд (polling)
- **localStorage** как fallback
- Кнопка «Синхронизировать» в настройках

### Тост-уведомления
- По центру, чёрный фон, белый текст, 4 секунды
